<p>Aspose.OCR für .NET ist eine robuste optische Zeichenerkennungs-API. Entwickler können ihren Anwendungen problemlos OCR-Funktionalitäten hinzufügen. Die API ist erweiterbar, einfach zu verwenden, kompakt und bietet einen einfachen Satz von Klassen zur Steuerung der Zeichenerkennung. Es unterstützt häufig verwendete Bildformate und bietet Funktionen wie das Lesen mehrerer Zeichen und Schriftarten aus Bildern, fette und kursive Stile, Rauschunterdrückungsfilter, Scannen des gesamten Bildes oder eines Teils des Bildes und vieles mehr. Darüber hinaus hat API die Fähigkeit, OCR unter Verwendung der GPU anstelle des Kernprozessors durchzuführen.</p>

<h2 class="h2title">Erfassen Sie Text aus Bildern</h2>

<p>Fügen Sie in nur 2 Codezeilen OCR-Funktionalität zu Ihren .NET-Anwendungen hinzu, um Text aus Bildern zu erfassen. Es ist wirklich so einfach!</p>

<div id="code" class="codeblock">

<h3>Einzeilige OCR-Erkennung – C#</h3>

<pre><code class="cs">// Eine Instanz von AsposeOcr initialisieren
AsposeOcr api = new AsposeOcr();

// Bild erkennen
string result = api.RecognizeImage(dataDir + "Sampleocr.bmp");

// Zeigt den erkannten Text an
Console.WriteLine(result);</code></pre>

</div>

</div>

   <div class="col-lg-12">
    <h2 class="h2title">
     Führen Sie OCR für Bilder durch
    </h2>
    <p>
     Aspose.OCR für .NET ermöglicht den Entwicklern, Text und zugehörige Informationen wie die Position des Textes aus bestimmten Teilen eines Bildes zu extrahieren. Diese Funktion ermöglicht die schnelle Durchführung des OCR-Vorgangs bei Dokumentenscans, die einer ähnlichen Struktur folgen. Die OCR-Bibliothek führt die Operation auf dem Bild mit mehreren Zeilen oder einer einzelnen Zeile durch. Hier ist der Code für die Einzelerkennung.
    </p>
    <div class="codeblock" id="code">
     <h3>
      OCR für Bild durchführen – C#
     </h3>
     <pre><code class="cs">// Der Pfad zum Dokumentenverzeichnis.
     
  string path = "dirPath/";

// Eine Instanz von AsposeOcr initialisieren
var api = new AsposeOcr();

// Bild erkennen
string result = api.RecognizeLine(path + "sample_line.png");

// Einzeiligen erkannten Text drucken
Console.WriteLine(result);</code></pre>
    </div>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">

    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Mehrsprachige Unterstützung
    </h2>
    <p>
     Aspose.OCR für .NET unterstützt verschiedene Sprachen wie Englisch, Französisch, Spanisch und Portugiesisch. Mit zukünftigen Releases der API sollen weitere Sprachen unterstützt werden. Die Durchführung eines OCR-Vorgangs ist für alle Sprachen gleich, wie z. B. nach einigen Zeilen, um einen OCR-Vorgang an einem Bild mit spanischem Text durchzuführen.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Unterstützung der automatischen OCR-Spracherkennung – C#
     </h3>
     <pre><code class="cs">// Verzeichnispfad der Dokumente

string dirPath = "directorypath/";

// Eine Instanz initialisieren
AsposeOcr ocrLanguage = new AsposeOcr();

// Bild erkennen
string spanishText = ocrLanguage.RecognizeImage(dirPath + "SpanischOCR.bmp");

// Den erkannten Text drucken
Console.WriteLine(spanishText);</code></pre>
