<p align="justify">
 Aspose.OCR pour Java est une API OCR autonome pour les applications Java tout en permettant aux développeurs d'effectuer une reconnaissance optique de caractères sur les types d'images couramment utilisés. Il fournit un ensemble simple de classes pour contrôler la reconnaissance de caractères pour divers caractères de langue. Les développeurs peuvent facilement lire des caractères dans plusieurs polices, des images pivotées, des images entières ou des parties d'images. De plus, l'API a la capacité d'effectuer l'OCR via le GPU au lieu du processeur principal.
</p>

    </h2>

    <div class="codeblock" id="code">
     <h3>
      Effectuer l'OCR sur des zones sélectionnées - Java
     </h3>
     <pre><code class="java">//Créer une instance d'API

AsposeOCR api = new AsposeOCR();

//Préparer des rectangles avec des textes.

ArrayList rectArray = new ArrayList();

rectArray.add(new Rectangle(138, 352, 2033, 537));

rectArray.add(new Rectangle(147, 890, 2033, 1157));

String result = api.RecognizePage("srcImage.png", rectArray);

System.out.println("Result with rect: " + result);</code></pre>
