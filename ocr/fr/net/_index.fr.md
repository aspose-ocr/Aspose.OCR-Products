<p>Aspose.OCR pour .NET est une API robuste de reconnaissance optique de caractères. Les développeurs peuvent facilement ajouter des fonctionnalités OCR dans leurs applications. L'API est extensible, facile à utiliser, compacte et fournit un ensemble simple de classes pour contrôler la reconnaissance des caractères. Il prend en charge les formats d'image couramment utilisés et fournit des fonctionnalités telles que la lecture de plusieurs caractères et polices à partir d'images, les styles gras et italique, les filtres de suppression du bruit, la numérisation de l'image entière ou de n'importe quelle partie de l'image et bien plus encore. De plus, l'API a la capacité d'effectuer l'OCR en utilisant le GPU au lieu du processeur principal.</p>

<h2 class="h2title">Capturer du texte à partir d'images</h2>

<p>En seulement 2 lignes de code, ajoutez la fonctionnalité OCR à vos applications .NET pour capturer du texte à partir d'images. C'est vraiment aussi simple!</p>

<div id="code" class="codeblock">

<h3>Reconnaissance OCR sur une seule ligne - C#</h3>

<pre><code class="cs">// Initialise une instance de AsposeOcr
AsposeOcr api = new AsposeOcr();

// Reconnaître l'image
string result = api.RecognizeImage(dataDir + "Sampleocr.bmp");

// Affiche le texte reconnu
Console.WriteLine(result);</code></pre>

</div>

</div>

   <div class="col-lg-12">
    <h2 class="h2title">
     Effectuer l'OCR sur les images
    </h2>
    <p>
     Aspose.OCR pour .NET permet aux développeurs d'extraire du texte et des informations connexes telles que l'emplacement du texte à partir de parties spécifiques d'une image. Cette fonctionnalité permet d'effectuer rapidement l'opération OCR sur des numérisations de documents qui suivent une structure similaire. La bibliothèque OCR effectue l'opération sur l'image comportant plusieurs lignes ou une seule ligne. Voici code le code pour la reconnaissance unique.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Effectuer l'OCR sur l'image - C#
     </h3>
     <pre><code class="cs">// Le chemin d'accès au répertoire des documents.
     
  string path = "dirPath/";

// Initialise une instance de AsposeOcr
var api = new AsposeOcr();

// Reconnaître l'image
string result = api.RecognizeLine(path + "sample_line.png");

// Imprimer le texte reconnu sur une seule ligne
Console.WriteLine(result);</code></pre>
    </div>
   </div>
   <div class="col-lg-12">

    <h2 class="h2title">
     Prise en charge multilingue

D'autres langues seront prises en charge avec les futures versions de l'API. L'exécution de l'opération OCR est la même pour toutes les langues, par exemple en suivant quelques lignes pour effectuer l'opération OCR sur une image contenant du texte espagnol.
    </p>

      Prise en charge de la détection automatique du langage OCR - C#

string dirPath = "directorypath/";

// Initialise une instance
AsposeOcr ocrLanguage = new AsposeOcr();

// Reconnaître l'image
string spanishText = ocrLanguage.RecognizeImage(dirPath + "EspagnolOCR.bmp");

// Affiche le texte reconnu
Console.WriteLine(spanishText);</code></pre>
