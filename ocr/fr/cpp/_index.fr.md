<p align="justify">
 Aspose.OCR pour C++ est une API de reconnaissance optique de caractères professionnelle robuste et fiable. Les développeurs peuvent facilement ajouter des opérations OCR dans leurs applications logicielles. La bibliothèque OCR peut également fonctionner hors ligne pour les applications professionnelles. De plus, cette API OCR de prémisse peut effectuer l'OCR en utilisant le GPU au lieu du processeur principal. La principale fonctionnalité de l'API est la détection de fragments de texte qui permet de reconnaître l'en-tête et les paragraphes des pages contenant des images ou des tableaux. Il prend en charge les formats couramment utilisés pour lire les caractères et les polices ayant différents styles, les filtres de suppression du bruit, la reconnaissance de la page entière ou d'une seule ligne.
</p>
