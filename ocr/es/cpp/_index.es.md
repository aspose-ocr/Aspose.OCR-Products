<p align="justify">
 Aspose.OCR para C++ es una API de reconocimiento óptico de caracteres profesional robusta y fiable. Los desarrolladores pueden agregar fácilmente operaciones de OCR en sus aplicaciones de software. La biblioteca OCR también puede funcionar sin conexión para aplicaciones comerciales. Además, esta API OCR de premisa puede realizar OCR utilizando la GPU en lugar del procesador central. La característica principal de la API es la detección de fragmentos de texto que permite reconocer encabezados y párrafos para páginas que tienen imágenes o tablas. Es compatible con formatos de uso común para leer caracteres y fuentes con diferentes estilos, filtros de eliminación de ruido, reconocimiento de la página completa o solo una línea.
</p>
