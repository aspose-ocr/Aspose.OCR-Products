<p align="justify">
 Aspose.OCR för C++ är en robust och pålitlig professionell API för optisk teckenigenkänning. Utvecklare kan enkelt lägga till OCR-operationer i sina program. OCR-biblioteket kan även fungera offline för affärsapplikationer. Dessutom kan denna förutsättning OCR API utföra OCR med GPU istället för kärnprocessorn. Den viktigaste API-funktionen är detektering av textfragment som gör det möjligt att känna igen rubriker och stycken för sidor med bilder eller tabeller. Den stöder vanliga format för att läsa tecken och teckensnitt med olika stilar, filter för brusborttagning, igenkänning av hela sidan eller bara en rad.
</p>
