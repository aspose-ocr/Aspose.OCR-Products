 Aspose.OCR for Java는 개발자가 일반적으로 사용되는 이미지 유형에 대해 광학 문자 인식을 수행할 수 있도록 하는 Java 애플리케이션용 독립 실행형 OCR API입니다. 다양한 언어 문자에 대한 문자 인식을 제어하는 간단한 클래스 세트를 제공합니다. 개발자는 여러 글꼴, 회전된 이미지, 전체 이미지 또는 이미지의 일부로 된 문자를 쉽게 읽을 수 있습니다. 또한 API는 코어 프로세서가 아닌 GPU를 통해 OCR을 수행할 수 있는 기능이 있습니다.
