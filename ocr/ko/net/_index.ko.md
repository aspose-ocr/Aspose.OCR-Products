<h2 class="h2title">이미지에서 텍스트 캡처</h2>

<p>단 2줄의 코드로 .NET 애플리케이션에 OCR 기능을 추가하여 이미지에서 텍스트를 캡처합니다. 정말 간단합니다!</p>

<div id="code" class="codeblock">

<h3>한 줄 OCR 인식 - C#</h3>

<pre><code class="cs">// AsposeOcr 인스턴스 초기화
AsposeOcr api = new AsposeOcr();

// 이미지 인식
string result = api.RecognizeImage(dataDir + "Sampleocr.bmp");

// 인식된 텍스트 표시
Console.WriteLine(result);</code></pre>

</div>

</div>

   <div class="col-lg-12">
    <h2 class="h2title">
     이미지에서 OCR 수행
    </h2>
    <p>
     .NET용 Aspose.OCR을 사용하면 개발자가 이미지의 특정 부분에서 텍스트 및 텍스트 위치와 같은 관련 정보를 추출할 수 있습니다. 이 기능을 사용하면 유사한 구조를 따르는 문서 스캔에서 OCR 작업을 빠르게 수행할 수 있습니다. OCR 라이브러리는 여러 줄 또는 한 줄의 이미지에 대한 작업을 수행합니다. 다음은 단일 인식을 위한 코드입니다.
    </p>
    <div class="codeblock" id="code">
     <h3>
      이미지에서 OCR 수행 - C#
     </h3>
     <pre><code class="cs">// 문서 디렉토리의 경로.
     
  string path = "dirPath/";

// AsposeOcr 인스턴스 초기화
var api = new AsposeOcr();

// 이미지 인식
string result = api.RecognizeLine(path + "sample_line.png");

// 인식된 한 줄의 텍스트를 인쇄합니다.
Console.WriteLine(result);</code></pre>
    </div>

    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     다국어 지원
    </h2>
    <p>
     .NET용 Aspose.OCR은 영어, 프랑스어, 스페인어 및 포르투갈어와 같은 다양한 언어를 지원합니다. API의 향후 릴리스에서는 더 많은 언어가 지원될 예정입니다. OCR 작업 수행은 스페인어 텍스트가 있는 이미지에 대해 다음 몇 줄만 OCR 작업을 수행하는 것과 같이 모든 언어에 대해 동일합니다.
    </p>
    <div class="codeblock" id="code">

      OCR 언어 자동 감지 지원 - C#

string dirPath = "directorypath/";

// 인스턴스 초기화
AsposeOcr ocrLanguage = new AsposeOcr();

// 이미지 인식
string spanishText = ocrLanguage.RecognizeImage(dirPath + "스페인의OCR.bmp");
