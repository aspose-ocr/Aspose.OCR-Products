<h2 class="h2title">從圖像中捕獲文本</h2>

<p>只需 2 行代碼，即可將 OCR 功能添加到您的 .NET 應用程序以從圖像中捕獲文本。真的就是這麼簡單！</p>

<div id="code" class="codeblock">

<h3>單行 OCR 識別 - C#</h3>

<pre><code class="cs">// 初始化 AsposeOcr 的一個實例
AsposeOcr api = new AsposeOcr();

// 識別圖像
string result = api.RecognizeImage(dataDir + "Sampleocr.bmp");

// 顯示識別的文本
Console.WriteLine(result);</code></pre>

</div>

</div>

   <div class="col-lg-12">
    <h2 class="h2title">
     對圖像執行 OCR
    </h2>
    <p>
     Aspose.OCR for .NET 允許開發人員從圖像的特定部分提取文本和相關信息，例如文本的位置。此功能允許對遵循類似結構的文檔掃描快速執行 OCR 操作。 OCR 庫對具有多行或單行的圖像執行操作。這是用於單識別的代碼。
    </p>
    <div class="codeblock" id="code">
     <h3>
      對圖像執行 OCR - C#
     </h3>
     <pre><code class="cs">// 文檔目錄的路徑。
     
  string path = "dirPath/";

// 初始化 AsposeOcr 的一個實例
var api = new AsposeOcr();

// 識別圖像
string result = api.RecognizeLine(path + "sample_line.png");

// 打印單行識別文本
Console.WriteLine(result);</code></pre>
    </div>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     多種字體樣式支持

    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     多語言支持
    </h2>
    <p>
     Aspose.OCR for .NET 支持不同的語言，例如英語、法語、西班牙語和葡萄牙語。未來的 API 版本將支持更多語言。對所有語言執行 OCR 操作都是相同的，例如以下幾行對具有西班牙語文本的圖像執行 OCR 操作。

    <div class="codeblock" id="code">

string dirPath = "directorypath/";

// 初始化一個實例
AsposeOcr ocrLanguage = new AsposeOcr();

string spanishText = ocrLanguage.RecognizeImage(dirPath + "西班牙語OCR.bmp");

// 打印識別的文本
Console.WriteLine(spanishText);</code></pre>
