<p align="justify">
 Aspose.OCR for C++ 是一個強大可靠的專業光學字符識別 API。開發人員可以輕鬆地在他們的軟件應用程序中添加 OCR 操作。 OCR 庫也可以離線工作以用於業務應用程序。此外，這個前提 OCR API 可以使用 GPU 而不是核心處理器來執行 OCR。主要 API 功能是文本片段檢測，可以識別具有圖片或表格的頁面的標題和段落。它支持用於讀取具有不同樣式的字符和字體的常用格式、去噪過濾器、識別整個頁面或僅單行。
