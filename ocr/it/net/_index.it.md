<p>Aspose.OCR per .NET è una robusta API di riconoscimento ottico dei caratteri. Gli sviluppatori possono aggiungere facilmente funzionalità OCR nelle loro applicazioni. L'API è estensibile, facile da usare, compatta e fornisce un semplice insieme di classi per il controllo del riconoscimento dei caratteri. Supporta i formati di immagine comunemente usati e fornisce funzionalità come la lettura di più caratteri e font dalle immagini, stili in grassetto e corsivo, filtri di rimozione del rumore, scansione dell'intera immagine o di qualsiasi parte dell'immagine e molto altro. Inoltre, l'API ha la capacità di eseguire l'OCR utilizzando la GPU anziché il core processor.</p>

<h2 class="h2title">Cattura il testo dalle immagini</h2>

<p>In sole 2 righe di codice, aggiungi la funzionalità OCR alle tue applicazioni .NET per acquisire testo dalle immagini. È davvero così semplice!</p>

<div id="code" class="codeblock">

<h3>Riconoscimento OCR a riga singola - C#</h3>

<pre><code class="cs">// Inizializza un'istanza di AsposeOcr
AsposeOcr api = new AsposeOcr();

// Riconosci l'immagine
string result = api.RecognizeImage(dataDir + "Sampleocr.bmp");

// Visualizza il testo riconosciuto
Console.WriteLine(result);</code></pre>

</div>

</div>

   <div class="col-lg-12">
    <h2 class="h2title">
     Eseguire l'OCR sulle immagini
    </h2>
    <p>
     Aspose.OCR per .NET consente agli sviluppatori di estrarre testo e informazioni correlate come la posizione del testo da parti specifiche di un'immagine. Questa funzione consente di eseguire rapidamente l'operazione OCR su scansioni di documenti che seguono una struttura simile. La libreria OCR esegue l'operazione sull'immagine con più righe o una singola riga. Ecco codificare il codice per il riconoscimento singolo.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Esegui OCR sull'immagine - C#
     </h3>
     <pre><code class="cs">// Il percorso della directory dei documenti.
     
  string path = "dirPath/";

// Inizializza un'istanza di AsposeOcr
var api = new AsposeOcr();

// Riconosci l'immagine
string result = api.RecognizeLine(path + "sample_line.png");

// Stampa il testo riconosciuto a riga singola
Console.WriteLine(result);</code></pre>
    </div>
