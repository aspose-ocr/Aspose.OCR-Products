<p align="justify">
 Aspose.OCR per C++ è un'API di riconoscimento ottico dei caratteri professionale robusta e affidabile. Gli sviluppatori possono facilmente aggiungere operazioni OCR nelle loro applicazioni software. La libreria OCR può funzionare offline anche per le applicazioni aziendali. Inoltre, questa API OCR premessa può eseguire l'OCR utilizzando la GPU anziché il core processor. La principale caratteristica dell'API è il rilevamento dei frammenti di testo che consente di riconoscere l'intestazione e i paragrafi per le pagine con immagini o tabelle. Supporta formati comunemente usati per leggere caratteri e font con stili diversi, filtri per la rimozione del rumore, riconoscimento dell'intera pagina o solo di una singola riga.
</p>
