Com Java OCR e apenas algumas linhas de código, você pode criar um aplicativo completo que converte uma imagem WBMP em um documento Text:

{{% /blocks/products/pf/agp/text %}}

+ Crie uma instância da classe AsposeOcr
+ Chamar o método AsposeOCR.RecognizePage
+ Passe o caminho do arquivo WBMP como parâmetro
+ AsposeOCR.RecognizePage retorna uma String ou arquivo do tipo Text
