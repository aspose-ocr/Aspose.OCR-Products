<p align="justify">
 Aspose.OCR para Java é uma API de OCR independente para aplicativos Java, permitindo que os desenvolvedores executem o reconhecimento óptico de caracteres em tipos de imagem comumente usados. Ele fornece um conjunto simples de classes para controlar o reconhecimento de caracteres de vários idiomas. Os desenvolvedores podem ler facilmente caracteres em várias fontes, imagens giradas, imagens inteiras ou partes de imagens. Além disso, a API tem a capacidade de executar OCR por meio da GPU em vez do processador central.
</p>
