<p align="justify">
 Aspose.OCR para C++ é uma API de reconhecimento óptico de caracteres profissional robusta e confiável. Os desenvolvedores podem facilmente adicionar operações de OCR em seus aplicativos de software. A biblioteca OCR também pode funcionar offline para aplicativos de negócios. Além disso, essa API de OCR premissa pode executar OCR usando a GPU em vez do processador central. O principal recurso da API é a detecção de fragmentos de texto que permite reconhecer cabeçalhos e parágrafos para páginas com imagens ou tabelas. Ele suporta formatos comumente usados para leitura de caracteres e fontes com estilos diferentes, filtros de remoção de ruído, reconhecimento de toda a página ou apenas uma linha.
</p>
