<p>Aspose.OCR para .NET é uma API robusta de reconhecimento óptico de caracteres. Os desenvolvedores podem adicionar facilmente funcionalidades de OCR em seus aplicativos. A API é extensível, fácil de usar, compacta e fornece um conjunto simples de classes para controlar o reconhecimento de caracteres. Ele suporta formatos de imagem comumente usados e fornece funcionalidades como leitura de vários caracteres e fontes de imagens, estilos de negrito e itálico, filtros de remoção de ruído, digitalização de toda a imagem ou de qualquer parte da imagem e muito mais. Além disso, a API tem a capacidade de executar OCR usando a GPU em vez do processador central.</p>

<h2 class="h2title">Capturar texto de imagens</h2>

<p>Em apenas 2 linhas de código, adicione a funcionalidade OCR aos seus aplicativos .NET para capturar texto de imagens. É realmente tão simples!</p>

<div id="code" class="codeblock">

<h3>Reconhecimento OCR de linha única - C#</h3>

<pre><code class="cs">// Inicializa uma instância de AsposeOcr
AsposeOcr api = new AsposeOcr();

// Reconhece a imagem
string result = api.RecognizeImage(dataDir + "Sampleocr.bmp");

// Exibe o texto reconhecido
Console.WriteLine(result);</code></pre>

</div>

</div>

   <div class="col-lg-12">
    <h2 class="h2title">
     Executar OCR em Imagens
    </h2>
    <p>
     O Aspose.OCR para .NET permite que os desenvolvedores extraiam texto e informações relacionadas, como a localização do texto de partes específicas de uma imagem. Esse recurso permite realizar a operação de OCR rapidamente em digitalizações de documentos que seguem uma estrutura semelhante. A biblioteca OCR executa a operação na imagem com várias linhas ou uma única linha. Aqui está o código do código para reconhecimento único.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Executar OCR na imagem - C#
     </h3>
     <pre><code class="cs">// O caminho para o diretório de documentos.
     
  string path = "dirPath/";

// Inicializa uma instância de AsposeOcr
var api = new AsposeOcr();

// Reconhece a imagem
string result = api.RecognizeLine(path + "sample_line.png");

// Imprime texto reconhecido de linha única
Console.WriteLine(result);</code></pre>
    </div>
   </div>

string dirPath = "directorypath/";

// Inicializa uma instância
AsposeOcr ocrLanguage = new AsposeOcr();
