<p>Aspose.OCR dla platformy .NET to niezawodny interfejs API do optycznego rozpoznawania znaków. Deweloperzy mogą łatwo dodawać funkcjonalności OCR w swoich aplikacjach. API jest rozszerzalne, łatwe w użyciu, kompaktowe i zapewnia prosty zestaw klas do kontrolowania rozpoznawania znaków. Obsługuje powszechnie używane formaty obrazów i zapewnia funkcje, takie jak odczytywanie wielu znaków i czcionek z obrazów, style pogrubienia i kursywy, filtry usuwania szumów, skanowanie całego obrazu lub dowolnej części obrazu i wiele innych. Co więcej, API ma możliwość wykonywania OCR przy użyciu GPU zamiast procesora rdzeniowego.</p>

<h2 class="h2title">Przechwytywanie tekstu z obrazów</h2>

<p>W zaledwie 2 linijkach kodu dodaj funkcję OCR do aplikacji .NET, aby przechwytywać tekst z obrazów. To jest naprawdę takie proste!</p>

<div id="code" class="codeblock">

<h3>Rozpoznawanie OCR w jednym wierszu — C#</h3>

<pre><code class="cs">// Zainicjuj instancję AsposeOcr
AsposeOcr api = new AsposeOcr();

// Rozpoznaj obraz
string result = api.RecognizeImage(dataDir + "Sampleocr.bmp");

// Wyświetl rozpoznany tekst
Console.WriteLine(result);</code></pre>

</div>

</div>

   <div class="col-lg-12">
    <h2 class="h2title">
     Wykonaj OCR na obrazach
    </h2>
    <p>
     Aspose.OCR dla platformy .NET umożliwia programistom wyodrębnienie tekstu i powiązanych informacji, takich jak lokalizacja tekstu, z określonych części obrazu. Ta funkcja umożliwia szybkie wykonanie operacji OCR na skanach dokumentów o podobnej strukturze. Biblioteka OCR wykonuje operację na obrazie posiadającym wiele linii lub jedną linię. Oto kod do pojedynczego rozpoznania.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Wykonaj OCR na obrazie — C#
     </h3>
     <pre><code class="cs">// Ścieżka do katalogu dokumentów.
     
  string path = "dirPath/";

// Zainicjuj instancję AsposeOcr
var api = new AsposeOcr();

// Rozpoznaj obraz
string result = api.RecognizeLine(path + "sample_line.png");

// Drukuj rozpoznany tekst w jednym wierszu
Console.WriteLine(result);</code></pre>
    </div>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Muti-czcionki i amp; Obsługa stylów

    </p>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Obsługa wielu języków
    </h2>
    <p>
     Aspose.OCR for .NET obsługuje różne języki, takie jak angielski, francuski, hiszpański i portugalski. Więcej języków będzie obsługiwanych w przyszłych wersjach API. Wykonywanie operacji OCR jest takie samo dla wszystkich języków, na przykład kilka linijek wykonuje operację OCR na obrazie z tekstem w języku hiszpańskim.
    </p>

string dirPath = "directorypath/";

// Zainicjuj instancję
AsposeOcr ocrLanguage = new AsposeOcr();

// Rozpoznaj obraz
string spanishText = ocrLanguage.RecognizeImage(dirPath + "hiszpańskiOCR.bmp");

// Wydrukuj rozpoznany tekst
Console.WriteLine(spanishText);</code></pre>
