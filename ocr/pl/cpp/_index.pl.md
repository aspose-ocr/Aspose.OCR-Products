<p align="justify">
 Aspose.OCR dla C++ to solidny i niezawodny interfejs API do optycznego rozpoznawania znaków. Deweloperzy mogą łatwo dodawać operacje OCR w swoich aplikacjach. Biblioteka OCR może również pracować w trybie offline dla aplikacji biznesowych. Co więcej, ten wstępny interfejs API OCR może wykonywać OCR przy użyciu GPU zamiast procesora rdzeniowego. Główną funkcją API jest wykrywanie fragmentów tekstu, które umożliwia rozpoznanie nagłówka i akapitów na stronach zawierających obrazy lub tabele. Obsługuje powszechnie używane formaty do odczytywania znaków i czcionek o różnych stylach, filtry usuwania szumów, rozpoznawanie całej strony lub tylko jednej linii.
</p>
