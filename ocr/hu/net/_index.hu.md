<p>Az Aspose.OCR for .NET egy robusztus optikai karakterfelismerő API. A fejlesztők egyszerűen hozzáadhatnak OCR-funkciókat alkalmazásaikhoz. Az API bővíthető, könnyen használható, kompakt, és egyszerű osztálykészletet biztosít a karakterfelismerés vezérléséhez. Támogatja a gyakran használt képformátumokat, és olyan funkciókat biztosít, mint például több karakter és betűtípus beolvasása a képekről, félkövér és dőlt stílusok, zajeltávolító szűrők, a teljes kép vagy a kép bármely részének beolvasása és még sok más. Ezen túlmenően, az API képes OCR-t végrehajtani a központi processzor helyett a GPU használatával.</p>

<h2 class="h2title">Szöveg rögzítése képekből</h2>

<p>Mindössze 2 sornyi kóddal adjon hozzá OCR funkciót .NET-alkalmazásaihoz, hogy szöveget rögzítsen képekből. Ez tényleg ilyen egyszerű!</p>

<div id="code" class="codeblock">

<h3>Egysoros OCR felismerés - C#</h3>

<pre><code class="cs">// Az AsposeOcr egy példányának inicializálása
AsposeOcr api = new AsposeOcr();

// Kép felismerése
string result = api.RecognizeImage(dataDir + "Sampleocr.bmp");

// A felismert szöveg megjelenítése
Console.WriteLine(result);</code></pre>

</div>

</div>

   <div class="col-lg-12">
    <h2 class="h2title">
     Végezze el az OCR-t a képeken
    </h2>
    <p>
     Az Aspose.OCR for .NET lehetővé teszi a fejlesztők számára, hogy szöveget és kapcsolódó információkat, például a szöveg helyét a kép bizonyos részeiből bontsa ki. Ez a funkció lehetővé teszi az OCR-művelet gyors végrehajtását a hasonló szerkezetet követő dokumentumok szkennelésekor. Az OCR könyvtár a műveletet a többsoros vagy egyetlen soros képen hajtja végre. Itt található az egyszeri felismerés kódja.
    </p>
    <div class="codeblock" id="code">
     <h3>
      OCR végrehajtása a képen - C#
     </h3>
     <pre><code class="cs">// A dokumentumok könyvtárának elérési útja.
     
  string path = "dirPath/";

// Az AsposeOcr egy példányának inicializálása
var api = new AsposeOcr();

// Kép felismerése
string result = api.RecognizeLine(path + "sample_line.png");

// Egysoros felismert szöveg nyomtatása
Console.WriteLine(result);</code></pre>
    </div>
