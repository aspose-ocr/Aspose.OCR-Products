<p align="justify">
 Az Aspose.OCR for C++ egy robusztus és megbízható professzionális optikai karakterfelismerő API. A fejlesztők könnyen hozzáadhatnak OCR-műveleteket szoftveralkalmazásaikhoz. Az OCR-könyvtár offline módban is működhet üzleti alkalmazásokhoz. Ezenkívül ez a feltételezett OCR API képes OCR-t végrehajtani a központi processzor helyett a GPU használatával. Az API fő funkciója a szövegtöredék-észlelés, amely lehetővé teszi a képekkel vagy táblázatokkal rendelkező oldalak fejléceinek és bekezdéseinek felismerését. Támogatja az általánosan használt formátumokat a különböző stílusú karakterek és betűtípusok olvasásához, zajeltávolító szűrőket, a teljes oldal vagy csak egyetlen sor felismerését.
</p>
