<p align="justify">
 Az Aspose.OCR for Java egy önálló OCR API Java alkalmazásokhoz, miközben lehetővé teszi a fejlesztők számára, hogy optikai karakterfelismerést hajtsanak végre a gyakran használt képtípusokon. Egyszerű osztálykészletet biztosít a különböző nyelvi karakterek karakterfelismerésének vezérléséhez. A fejlesztők könnyedén beolvashatják a karaktereket több betűtípussal, elforgatott képeket, teljes képeket vagy képrészleteket. Ezen túlmenően az API képes OCR végrehajtására GPU-n keresztül a magprocesszor helyett.
</p>
