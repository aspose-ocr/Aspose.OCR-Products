<p align="justify">
 Aspose.OCR untuk Java adalah API OCR yang berdiri sendiri untuk aplikasi Java sementara memungkinkan pengembang untuk melakukan pengenalan karakter optik pada jenis gambar yang umum digunakan. Ini menyediakan satu set kelas sederhana untuk mengontrol pengenalan karakter untuk berbagai karakter bahasa. Pengembang dapat dengan mudah membaca karakter dalam beberapa font, gambar yang diputar, seluruh gambar atau bagian dari gambar. Selain itu, API memiliki kemampuan untuk melakukan OCR melalui GPU alih-alih prosesor inti.
</p>
