<p align="justify">
 Aspose.OCR untuk C++ adalah API pengenalan karakter optik profesional yang tangguh dan andal. Pengembang dapat dengan mudah menambahkan operasi OCR dalam aplikasi perangkat lunak mereka. Pustaka OCR juga dapat bekerja secara offline untuk aplikasi bisnis. Selanjutnya, API OCR premis ini dapat melakukan OCR menggunakan GPU alih-alih prosesor inti. Fitur API utama adalah deteksi fragmen teks yang memungkinkan untuk mengenali header dan paragraf untuk halaman yang memiliki gambar atau tabel. Ini mendukung format yang umum digunakan untuk membaca karakter dan font yang memiliki gaya berbeda, filter penghilang noise, mengenali seluruh halaman, atau hanya satu baris.
</p>
