<p align="justify">
 Aspose.OCR for C++ 是一个强大可靠的专业光学字符识别 API。开发人员可以轻松地在他们的软件应用程序中添加 OCR 操作。 OCR 库也可以离线工作以用于业务应用程序。此外，这个前提 OCR API 可以使用 GPU 而不是核心处理器来执行 OCR。主要 API 功能是文本片段检测，可以识别具有图片或表格的页面的标题和段落。它支持用于读取具有不同样式的字符和字体的常用格式、去噪过滤器、识别整个页面或仅单行。
