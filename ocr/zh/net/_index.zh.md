<h2 class="h2title">从图像中捕获文本</h2>

<p>只需 2 行代码，即可将 OCR 功能添加到您的 .NET 应用程序以从图像中捕获文本。真的就是这么简单！</p>

<div id="code" class="codeblock">

<h3>单行 OCR 识别 - C#</h3>

<pre><code class="cs">// 初始化 AsposeOcr 的一个实例
AsposeOcr api = new AsposeOcr();

// 识别图像
string result = api.RecognizeImage(dataDir + "Sampleocr.bmp");

// 显示识别的文本
Console.WriteLine(result);</code></pre>

</div>

</div>

   <div class="col-lg-12">
    <h2 class="h2title">
     对图像执行 OCR
    </h2>
    <p>
     Aspose.OCR for .NET 允许开发人员从图像的特定部分提取文本和相关信息，例如文本的位置。此功能允许对遵循类似结构的文档扫描快速执行 OCR 操作。 OCR 库对具有多行或单行的图像执行操作。这是用于单识别的代码。
    </p>
    <div class="codeblock" id="code">
     <h3>
      对图像执行 OCR - C#
     </h3>
     <pre><code class="cs">// 文档目录的路径。
     
  string path = "dirPath/";

// 初始化 AsposeOcr 的一个实例
var api = new AsposeOcr();

// 识别图像
string result = api.RecognizeLine(path + "sample_line.png");

// 打印单行识别文本
Console.WriteLine(result);</code></pre>
    </div>
   </div>
   <div class="col-lg-12">

    </p>
   </div>
   <div class="col-lg-12">

     多语言支持

string dirPath = "directorypath/";

// 初始化一个实例
AsposeOcr ocrLanguage = new AsposeOcr();

string spanishText = ocrLanguage.RecognizeImage(dirPath + "西班牙语OCR.bmp");

// 打印识别的文本
Console.WriteLine(spanishText);</code></pre>
