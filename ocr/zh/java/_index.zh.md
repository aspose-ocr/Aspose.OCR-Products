 Aspose.OCR for Java 是用于 Java 应用程序的独立 OCR API，同时允许开发人员对常用图像类型执行光学字符识别。它提供了一组简单的类来控制各种语言字符的字符识别。开发人员可以轻松读取多种字体、旋转图像、整幅图像或部分图像的字符。此外，API 能够通过 GPU 而不是核心处理器执行 OCR。
