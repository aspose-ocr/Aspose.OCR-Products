<p align="justify">
 Aspose.OCR для C++ — это надежный профессиональный API для оптического распознавания символов. Разработчики могут легко добавлять операции OCR в свои программные приложения. Библиотека OCR также может работать в автономном режиме для бизнес-приложений. Кроме того, этот предварительный API OCR может выполнять распознавание символов с использованием графического процессора вместо основного процессора. Основной функцией API является обнаружение текстовых фрагментов, которое позволяет распознавать заголовок и абзацы для страниц, содержащих изображения или таблицы. Он поддерживает часто используемые форматы для чтения символов и шрифтов, имеющих различные стили, фильтры удаления шума, распознавание всей страницы или только одной строки.
</p>
