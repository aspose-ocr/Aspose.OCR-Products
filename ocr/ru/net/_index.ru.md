<p>Aspose.OCR для .NET — это надежный API для оптического распознавания символов. Разработчики могут легко добавлять функции OCR в свои приложения. API является расширяемым, простым в использовании, компактным и предоставляет простой набор классов для управления распознаванием символов. Он поддерживает часто используемые форматы изображений и предоставляет такие функции, как чтение нескольких символов и шрифтов из изображений, начертание полужирным шрифтом и курсивом, фильтры для удаления шума, сканирование всего изображения или любой его части и многое другое. Более того, API имеет возможность выполнять распознавание текста с использованием графического процессора вместо основного процессора.</p>

<h2 class="h2title">Захват текста из изображений</h2>

<p>Всего двумя строками кода добавьте в свои приложения .NET функции OCR для захвата текста с изображений. Это действительно так просто!</p>

<div id="code" class="codeblock">

<h3>Распознавание одной строки OCR — C#</h3>

<pre><code class="cs">// Инициализировать экземпляр AsposeOcr
AsposeOcr api = new AsposeOcr();

// Распознать изображение
string result = api.RecognizeImage(dataDir + "Sampleocr.bmp");

// Отображаем распознанный текст
Console.WriteLine(result);</code></pre>

</div>

</div>

   <div class="col-lg-12">
    <h2 class="h2title">
     Выполнение оптического распознавания символов на изображениях
    </h2>
    <p>
     Aspose.OCR для .NET позволяет разработчикам извлекать текст и связанную с ним информацию, такую как местоположение текста, из определенных частей изображения. Эта функция позволяет быстро выполнять операцию оптического распознавания символов на сканированных документах с аналогичной структурой. Библиотека OCR выполняет операцию над изображением, имеющим несколько строк или одну строку. Вот код кода для одиночного распознавания.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Выполнение оптического распознавания символов на изображении — C#
     </h3>
     <pre><code class="cs">// Путь к каталогу документов.
     
  string path = "dirPath/";

// Инициализировать экземпляр AsposeOcr
var api = new AsposeOcr();

// Распознать изображение
string result = api.RecognizeLine(path + "sample_line.png");

// Печать одной строки распознанного текста
Console.WriteLine(result);</code></pre>

Выполнение операции OCR одинаково для всех языков, например, следующие несколько строк выполняют операцию OCR для изображения с испанским текстом.
