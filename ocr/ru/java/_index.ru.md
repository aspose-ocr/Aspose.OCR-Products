<p align="justify">
 Aspose.OCR для Java — это автономный API-интерфейс OCR для приложений Java, который позволяет разработчикам выполнять оптическое распознавание символов на часто используемых типах изображений. Он предоставляет простой набор классов для управления распознаванием символов различных языков. Разработчики могут легко читать символы с несколькими шрифтами, повернутыми изображениями, целыми изображениями или частями изображений. Кроме того, API может выполнять распознавание текста с помощью графического процессора, а не основного процессора.
</p>
