<p align="justify">
 Aspose.OCR for C++ je robustní a spolehlivé profesionální rozhraní API pro optické rozpoznávání znaků. Vývojáři mohou snadno přidat operace OCR do svých softwarových aplikací. Knihovna OCR může fungovat i offline pro podnikové aplikace. Kromě toho může tento předpoklad OCR API provádět OCR pomocí GPU namísto základního procesoru. Hlavní funkcí API je detekce textových fragmentů, která umožňuje rozpoznat záhlaví a odstavce pro stránky s obrázky nebo tabulkami. Podporuje běžně používané formáty pro čtení znaků a písem s různými styly, filtry pro odstranění šumu, rozpoznání celé stránky nebo pouze jednoho řádku.
</p>
