<p>Aspose.OCR for .NET je robustní rozhraní API pro optické rozpoznávání znaků. Vývojáři mohou snadno přidat funkce OCR do svých aplikací. API je rozšiřitelné, snadno použitelné, kompaktní a poskytuje jednoduchou sadu tříd pro ovládání rozpoznávání znaků. Podporuje běžně používané obrazové formáty a poskytuje funkce jako čtení více znaků a písem z obrázků, styly tučného písma a kurzívy, filtry pro odstranění šumu, skenování celého obrázku nebo jakékoli části obrázku a mnoho dalšího. Kromě toho má API schopnost provádět OCR pomocí GPU namísto základního procesoru.</p>

<h2 class="h2title">Zachycení textu z obrázků</h2>

<p>V pouhých 2 řádcích kódu přidejte do svých aplikací .NET funkci OCR pro zachycení textu z obrázků. Je to opravdu tak jednoduché!</p>

<div id="code" class="codeblock">

<h3>Jednořádkové OCR rozpoznávání - C#</h3>

<pre><code class="cs">// Inicializuje instanci AsposeOcr
AsposeOcr api = new AsposeOcr();

// Rozpoznání obrázku
string result = api.RecognizeImage(dataDir + "Sampleocr.bmp");

// Zobrazí rozpoznaný text
Console.WriteLine(result);</code></pre>

</div>

</div>

   <div class="col-lg-12">
    <h2 class="h2title">
     Proveďte OCR na obrázcích
    </h2>
    <p>
     Aspose.OCR for .NET umožňuje vývojářům extrahovat text a související informace, jako je umístění textu z konkrétních částí obrázku. Tato funkce umožňuje rychlé provedení operace OCR u naskenovaných dokumentů, které mají podobnou strukturu. Knihovna OCR provádí operaci s obrazem, který má více řádků nebo jeden řádek. Zde je kód pro jednoduché rozpoznání.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Proveďte OCR na obrázku - C#
     </h3>
     <pre><code class="cs">// Cesta k adresáři dokumentů.
     
  string path = "dirPath/";

// Inicializuje instanci AsposeOcr
var api = new AsposeOcr();

// Rozpoznání obrázku
string result = api.RecognizeLine(path + "sample_line.png");

// Tisk jednoho řádku rozpoznaného textu
Console.WriteLine(result);</code></pre>
    </div>
