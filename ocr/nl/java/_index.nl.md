<p align="justify">
 Aspose.OCR voor Java is een stand-alone OCR-API voor Java-toepassingen, terwijl het de ontwikkelaars in staat stelt optische tekenherkenning uit te voeren op veelgebruikte afbeeldingstypen. Het biedt een eenvoudige reeks klassen om tekenherkenning voor verschillende taaltekens te regelen. Ontwikkelaars kunnen gemakkelijk tekens lezen in meerdere lettertypen, geroteerde afbeeldingen, hele afbeeldingen of delen van afbeeldingen. Bovendien heeft API de mogelijkheid om OCR uit te voeren via GPU in plaats van de kernprocessor.
</p>
